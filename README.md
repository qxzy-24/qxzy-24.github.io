## ✨ Features

- **Cybersecurity-Themed Design** — Dark CRT aesthetic with glitch effects, particle canvas background, and custom cursor animations.
- **Responsive Layout** — Fully optimized for desktop, tablet, and mobile viewports.
- **Interactive Elements** — 3D tilt cards, magnetic buttons, parallax scrolling, and text scramble effects on hover.
- **Typing Animation** — Dynamic role cycling in the hero section.
- **Scroll Reveal** — Content animates into view using IntersectionObserver.
- **Hidden Terminal** — Easter egg terminal accessible via the Konami code (`↑↑↓↓←→←→BA`). Supports commands like `whoami`, `skills`, `projects`, `hack`, and more.
- **Zero Dependencies** — Built entirely with vanilla HTML, CSS, and JavaScript. No frameworks.

## 🚀 Run Locally

```bash
git clone https://github.com/qxzy-24/qxzy-24.github.io.git
cd qxzy-24.github.io
```

Then open `index.html` in your browser, or use a local server:

```bash
npx serve .
```
